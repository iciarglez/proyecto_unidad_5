# PROYECTO_UNIDAD_5
Repositorio asociado al proyecto de la unidad 5.

Consultas SQL realizadas sobre la base de datos Shakila.


## Contenido

- Consultas de lógica SQL.
- Resolución de ejercicios de la unidad 5.


## Requisitos de la entrega

- Archivo README.md, que recoja los pasos seguidos durante el proyecto y el informe de tú análisis.
- Un esquema de la BBDD que te proporcionamos
- Archivo SQL con las consultas resueltas (Cada consulta debe estar identificada con su número y su enunciado como comentario)


## Pasos del proyecto

1. En primer lugar, cargamos la base de datos en DBeaver, después de descargar el archivo .sql de la plataforma ThePower.
Trabajamos en postgres, y establecemos por defecto "shakila", nuestra base de datos.

2. A continuación, analizamos la base de datos, a partir de la información contenida en Esquemas > public.
Seleccionando el botón derecho, View Diagram, tenemos un diagrama de todas las tablas, con las columnas, claves primarias y relaciones entre tablas.
Es fundamental revisar este diagrama para poder entender los datos con los que vamos a trabajar.
Lo exportamos, se trata del fichero shakila - public.png adjunto. Nos servirá de referencia a lo largo de todo el proyecto.

3. También podemos revisar las tablas con más detalle, en Tablas > ver tablas.
Esto nos permitirá ver las columnas de cada tabla y el tipo de dato.

4. Una vez hemos revisado la base de datos, respondemos a las preguntas planteadas.
Las soluciones están en el fichero PROYECTO_5_shakila_LogicaConsultasSQL.sql
Cada ejercicio contiene el enunciado, la resolución y anotaciones relacionadas con la resolución.
